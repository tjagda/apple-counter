Description
***********
Detect and count apples from an image using colour spaces
and segmentation algorithms

How To Run
***********
Be sure to have Python3, Numpy and OpenCV installed.

Then run the following command:
python3 test.py [imagefile]

Where [imagefile] is a compatible image file (possibly)
containing apple(s)
